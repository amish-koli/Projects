# Frontend Playground

This repository is a collection of **HTML, CSS, and SCSS projects** that showcase my work in frontend development. Each project is designed to improve my skills, experiment with different UI patterns, and create responsive, modern web designs.

---

## Projects Included
- Simple landing pages  
- Responsive navigation menus (including hamburger menus)  
- Forms and interactive components  
- Layout experiments using CSS Grid and Flexbox  
- Styling with SCSS and reusable variables/mixins  

> Each project is self-contained and can be viewed directly in the browser.

---

## Technologies Used
- **HTML5** – Semantic markup and structure  
- **CSS3** – Styling, animations, transitions, and responsive design  
- **SCSS** – Nested styling, variables, mixins, and modular CSS
